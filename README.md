# About the project
* A job listing application project developed with php + composer.
* Users can register, authenticate, and manage the job applications lifecycle.
* Data stored using PostgreSQL.
* Password reset feature.

# Entry point
* public/index.php

# ./App/Controllers
* All the methods for users and posts
* Password recovery feature
* views to render th html

# ./config/
* Define the database connection

# ./routes.php
* Endpoints definition

# ./Framework/
* Classes to be reused across the whole app

# ./script.sql
* sql seed file
